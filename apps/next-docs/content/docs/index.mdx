## Welcome to NovaUI

NovaUI is a modern React component library built with accessibility and developer experience in mind. It provides a comprehensive set of customizable components to help you build beautiful and functional user interfaces.

## Features

- **Accessibility First**: All components are built with ARIA compliance and keyboard navigation support
- **Fully Customizable**: Flexible theming system with support for custom variants
- **TypeScript Ready**: Complete TypeScript support with proper type definitions
- **Modern Stack**: Built with React 18+ and modern development practices
- **Tree Shakable**: Only import what you need to keep your bundle size small

## Getting Started

To get started with NovaUI, install the package using your preferred package manager:

```bash
npm install novaui
# or
pnpm add novaui
# or
yarn add novaui
```

### Basic Usage

Here's a simple example of how to use NovaUI components:

```tsx
import { Button, Card } from 'novaui'

function App() {
  return (
    <Card>
      <Card.Header>
        <Card.Title>Welcome to NovaUI</Card.Title>
      </Card.Header>
      <Card.Content>
        <p>Build amazing user interfaces with our component library.</p>
        <Button variant="primary">Get Started</Button>
      </Card.Content>
    </Card>
  )
}
```
